## Task Description

Automate the complete signup process on:
https://authorized-partner.vercel.app/

This automation covers the entire multi-step signup flow and completes the process automatically without manual intervention.

Java (JDK 21)

Selenium WebDriver 4

Maven

IntelliJ IDEA

Google Chrome
